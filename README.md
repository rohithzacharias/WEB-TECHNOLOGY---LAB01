# 🛒 TechZone Electronics Store

A simple and responsive **HTML5-based E-Commerce Electronics Store Website** developed as part of **Web Technology Laboratory - Experiment 1**.

## 📖 About the Project

TechZone Electronics Store is a basic e-commerce webpage designed using **HTML5** and **Internal CSS**. The website demonstrates the use of fundamental HTML elements and basic webpage styling while providing an attractive and user-friendly interface.

## ✨ Features

- Responsive webpage layout
- Attractive header section
- Navigation links
- Featured product image
- Product table
- Product categories using Description List
- Shopping process using Ordered Nested List
- Customer services using Unordered Nested List
- Email and Phone hyperlinks
- Internal and External hyperlinks
- Bookmark (Back to Top)
- Internal CSS styling

## 🛠️ Technologies Used

- HTML5
- CSS3 (Internal CSS)
